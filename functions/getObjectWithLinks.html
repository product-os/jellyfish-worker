Function getObjectWithLinks

  • getObjectWithLinks<PContract, TContract, TData>(logContext: LogContext, kernel: Kernel, session: AutumnDBSession, contract: PContract, typeContract: TypeContract): Promise<PContract>

  • Returns an object that will include all links referenced in evaluated fields in the type contract's schema.

    If no links are referenced in evaluated fields, the original object is returned immediately.
